Web tyleguides definition

Web site style guides cover a publication's visual and technical aspects, along with text. Many style guides are revised periodically to accommodate changes in conventions and usage.

Wikipedia
A styleguide is a tool to set standards in how the brand identity is communicated. Important in this definition is "visual and technical" and "revised periodically"

Do we need a styleguide for every project?

Back to the topic of this presentation ... Do we really need a styleguide for every project ...

Let's go back 2 years ago ...

So now, we're a team of 3 frontend developers ... working together on the same code. This guy is designing, and those 3 there are coding CSS/HTML
With this big client, a real estate company! They have a website with a search engine, some landing pages and other websites dedicated to estate properties.
They want the same design on all the website. 
              
                ul.nav-main .nav-sub .nav-sub-top, ul.nav-language .nav-sub .nav-sub-top {}
9000 lines of CSS, with no documentation, used accross 10 websites.

Needed to be solved?

  • Code consistency
  • Modular approach
  • Efficient collaboration
  • Living documentation
  • Easy testing
Main website with other landing page ... using the same CSS Collaboration with in-house developers, the need for a documentation (which stay up to date).
